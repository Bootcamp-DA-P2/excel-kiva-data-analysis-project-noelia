# 📊 Kiva Loans Analysis Dashboard

## 🧠 Project Overview

This project presents an exploratory data analysis of microfinance loans using the Kiva dataset, developed entirely in Microsoft Excel.

The objective is to understand how microloans are distributed globally by country, sector, loan size, duration, and borrower characteristics.

---

## 📸 Dashboard Preview

![Dashboard](docs/dashboard.png)

---

## 🎯 Objectives

* Analyze the distribution of loans across countries
* Identify patterns in loan size and duration
* Explore how loans are used across sectors
* Understand borrower characteristics, including gender distribution

---

## 🧹 Data Preparation

* Data imported and transformed using **Power Query**
* Removal of duplicates and basic data cleaning
* Data type validation and formatting
* Creation of new analytical variables:

  * **Loan Size Category** (Micro, Small, Medium, Large)
  * **Loan Term Category** (Short, Medium, Long)
* Separation and aggregation of borrower data by gender (male/female)

---

## 📊 Exploratory Data Analysis

* Application of **conditional formatting** to identify patterns and outliers
* Generation of **descriptive statistics** (mean, min, max, distribution)
* Identification of loan distribution trends

---

## 📈 Data Analysis

* Extensive use of **Pivot Tables** to calculate:

  * Total number of loans
  * Total loan amount
  * Average loan value
* Aggregation of data by:

  * Country
  * Sector
  * Loan size
  * Loan duration

---

## 📊 Dashboard & Visualization

* Creation of charts based on Pivot Tables:

  * Top countries by number of loans
  * Loan distribution by size
  * Loan duration analysis
  * Gender distribution by loan size
* Structured dashboard layout with clear visual hierarchy
* Use of consistent color palette and alignment

---

## 🎯 Key KPIs

* 💰 Total Loan Amount: **€357M**
* 📊 Total Number of Loans: **42,308**
* 📉 Average Loan Amount: **€8,443**
* 🌍 Number of Countries: **75**

## 📦 Loan Distribution

* Micro Loans: **50.5%**
* Small Loans: **26.2%**
* Medium Loans: **17.7%**
* Large Loans: **5.6%**

---

## ⚙️ Interactivity

* Implementation of **Slicers** to filter data dynamically
* Slicers connected across all Pivot Tables for full dashboard interaction

---

## 🚀 Key Insights

* The majority of loans are **microloans**, confirming Kiva’s focus on small-scale financing
* Loan duration is predominantly **medium to long term**
* A small number of countries concentrate a large proportion of loans
* **Female borrowers represent a higher share across all loan size categories**
